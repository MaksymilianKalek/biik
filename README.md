# Web3 blockchain application

This project contains a full-stack Solidity blockchain application with frontend implemented in Vue.js. It is a simple shopping list application where we can manipulate all of the products currently stored on the Ethereum blockchain.

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
